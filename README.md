Installing the Application<a name="TOP"></a>
===================

- - - - 
For the monolithic application,

### Install Packages ###

1. Run `npm i`
2. Run `npm start`

For the frontend application,

### Install Packages ###

1. Run `npm i`
2. Run `npm start`
